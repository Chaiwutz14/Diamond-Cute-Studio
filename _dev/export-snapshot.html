🛠️ สร้าง Static Snapshot

ดึงสินค้า + แกลเลอรีจาก Firestore แล้วสร้างไฟล์ JSON สำหรับวางใน /data/

ทำให้หน้าเว็บอ่านจากไฟล์ (ผ่าน CDN) แทน Firestore → รับคนได้ไม่จำกัด ไม่เปลืองโควต้าฟรี

ขั้นตอนถัดไป (อัปขึ้น GitHub):
  1. กด "สร้าง Snapshot" → ดาวน์โหลดทั้ง 2 ไฟล์
  2. ไปที่ repo บน GitHub.com → เข้าโฟลเดอร์ data
  3. กด Add file → Upload files → ลากไฟล์ products.json และ gallery.json ลงไป
  4. กด Commit changes → เสร็จ! หน้าเว็บจะอ่านจากไฟล์ทันที

ทำซ้ำทุกครั้งที่เพิ่ม/แก้/ลบสินค้า เพื่อให้ snapshot ตรงกับข้อมูลจริง
